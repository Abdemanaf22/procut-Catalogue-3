🚀 How to Update Your Product Catalogue

Your catalogue uses a simple products.json file. Here's how to update it when needed:

METHOD 1: Use This Admin Panel (Easiest)

1
Edit prices/names directly in the table — click the price and type a new value, then click Save.
2
Click "Export JSON" button at the top to download the updated products.json
3
Replace the file in your website folder: put the new products.json into the data/ folder.
4
Upload to GitHub — commit and push, GitHub Pages updates automatically in ~30 seconds.

METHOD 2: Direct JSON Edit (Advanced)

Open data/products.json in any text editor. Each product looks like:

{ "id": 1, "name": "Heavy Duty Rubber Mallet", "category": "Hand Tools", "code": "HT001A", "wholesalePrice": 120, "endUserPrice": 180, "priceDisplay": "From ₹120/-", "gst": 18, "hsn": "8205", "brand": "Procut" }

Change wholesalePrice and endUserPrice numbers. Save and upload.

📦 GitHub Pages Hosting (Free)

1
Go to github.com → Create account → New repository → Name it procut-catalogue
2
Upload ALL files: index.html, admin.html, assets/ folder, data/products.json
3
Settings → Pages → Source: Deploy from branch → main → / (root) → Save
4
Your site will be live at: https://yourusername.github.io/procut-catalogue/
